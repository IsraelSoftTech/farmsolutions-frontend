This directory contains logos for our partners. Please replace the placeholder files with actual logos from the official partner websites.

## Logo Specifications

- Format: PNG or SVG (preferably PNG with transparency)
- Minimum size: 400x200px for best quality
- Background: Transparent (preferred) or white
- File naming: Use lowercase with hyphens (as shown above)

## Adding Logos

1. Obtain official logos from partner websites or contact them directly
2. Ensure you have permission to use the logos
3. Resize and optimize logos if needed
4. Save files in this directory with the exact filenames listed above
5. The website will automatically display the new logos
